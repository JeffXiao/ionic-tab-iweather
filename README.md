# ionic-tab-iweather
A demo app that demonstrate a mobile weather using ionic</br>

1) Install ionic:</br>
$ npm install -g ionic cordova</br>
</br>
2) Install all the open source libraries:</br>
$ npm install</br>
or</br>
$ npm update</br>
</br>
3) Run the app:</br>
$ ionic serve -c</br>
$ ionic serve --lab -c</br>
